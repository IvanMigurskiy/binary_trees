# binary_trees

Данный репозиторий содержит набор реализаций различных деревьев поиска и вспомогательных классов для работы с ними, включая шаблонные классы сравнения, ошибки и структуру данных "отношение" (relation), позволяющую управлять множеством деревьев с разными типами и индексами.

## Структура проекта

- `Header.h` — основной заголовочный файл, содержащий реализации шаблонных классов и методов
- `laba2_tree.cpp` — пример использования данных структур и классов, демонстрирующий работу с ними

## Основные компоненты

- **Compare<T>** — интерфейс для сравнения элементов
- **IntCompare, StrCompare** — реализации сравнения для `int` и `std::string`
- **Шаблоны деревьев поиска**:
  - Бинарное дерево (`bin_tree`)
  - АВЛ-дерево (`AVL_tree`)
  - Splay-дерево (`splay_tree`)
  - Красно-черное дерево (`RBtree`)
  - 2-3-дерево (`tree_23`)
- **Класс `relation`** — управляет множеством деревьев с разными индексами, позволяет добавлять/удалять индексы и данные

## Использование

Пример использования показан в файле `laba2_tree.cpp`, где создаются деревья разных типов, добавляются данные, осуществляется поиск и удаление.
